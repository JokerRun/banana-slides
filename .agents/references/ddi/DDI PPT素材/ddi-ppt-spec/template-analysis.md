# DDI PPT 模板分析报告 (DDI_Brand-PPT-16x9-Template_2025)

本报告详细分析了 `DDI_Brand-PPT-16x9-Template_2025 (002).pptx` 模板的设计规范、布局类型及视觉元素。

## 1. 颜色方案 (Color Scheme)

模板采用了 **DDI Gray Series** 主题，以专业、稳重的深蓝色和灰色系为主，搭配明亮的橙黄色作为强调色。

| 类别 | HEX 值 | 说明 |
| :--- | :--- | :--- |
| **主背景/文本 1 (dk1)** | `#425563` | 深灰蓝色 (Dark Slate Gray)，用于主要文本和暗色背景 |
| **主背景/文本 2 (lt1)** | `#FFFFFF` | 纯白色，用于背景和反白文字 |
| **强调色 1 (Accent 1)** | `#FFB81C` | 琥珀色/橙黄色，用于重点强调、图标或点缀 |
| **强调色 2 (Accent 2)** | `#D9E1E2` | 浅灰青色，用于辅助图形或浅色背景块 |
| **强调色 3 (Accent 3)** | `#89B5D4` | 天蓝色，用于图表或辅助设计元素 |
| **强调色 4 (Accent 4)** | `#3A84B6` | 中蓝色，用于主要色块或链接 |
| **强调色 5 (Accent 5)** | `#236192` | 深蓝色 (dk2)，用于核心品牌色 |
| **强调色 6 (Accent 6)** | `#ACB0B7` | 灰色，用于次要文字或装饰 |
| **超链接 (Hyperlink)** | `#85888C` | 灰色 |
| **已访问链接** | `#F6A01F` | 橙色 |

## 2. 字体规范 (Typography)

| 类别 | 字体名称 | 建议字号 | 说明 |
| :--- | :--- | :--- | :--- |
| **标题 (Major Font)** | Century Gothic | 26pt - 36pt | 用于幻灯片主标题、章节标题 |
| **正文 (Minor Font)** | Arial | 14pt - 24pt | 用于正文内容、列表、脚注 |
| **代码/数据** | Arial | 10pt - 12pt | 用于图表注释或次要数据 |

*注：标题通常加粗 (`bold`) 处理。*

## 3. 设计元素 (Design Elements)

*   **Logo 位置：** DDI Logo 统一固定在幻灯片的 **右下角** (Bottom-Right)。
*   **装饰图形：** 
    *   大量使用灰色色块作为分节标识。
    *   部分布局包含带边框的图片框 (Picture Frame) 设计。
*   **页码与版权：** 
    *   左下角包含自动页码标识 `{#}`。
    *   底部中央或偏右位置包含版权声明：`©Development Dimensions International, Inc., 2025. All rights reserved.`
*   **视觉风格：** 整体风格简洁、商务，利用灰度差和蓝色调建立层级感。

## 4. 幻灯片布局类型 (Slide Layouts)

模板共包含 47 种预定义布局，核心类型如下：

*   **封面/标题页 (Title Slides):** 包含带照片的封面 (`Photo Title Slide`) 和纯文字封面 (`Gray Title Slide - Text Only`)。
*   **目录/章节页 (Section Dividers):** 提供了 16 种不同编号的灰色章节页 (`Title / Section Slide Gray`)。
*   **内容页 (Content):** 
    *   `Title and Text`: 标准标题+正文。
    *   `Title + two-column text`: 标题+双栏文字。
    *   `Title + Picture + Text`: 标题+图片+文字组合。
    *   `Title Left + Text Right`: 左右分割布局。
*   **图片焦点页 (Picture Focus):** 各种带边框或全屏的图片展示页 (`Picture Frame`)。
*   **特殊用途页:** `Polling or Quiz Slide` (投票/问答)、`Client quote or story` (客户证言/引用)。

基于模板中的 32 个示例幻灯片，其实际使用的布局顺序如下：

| 幻灯片序号 | 布局名称 | 功能/用途说明 |
| :--- | :--- | :--- |

| 2 | Gray Title Slide - Text Only | 纯文字封面或子标题页 |
| 3-11 | Title / Section Slide Gray | 章节过渡页 (各种变体 8, 10, 11, 12, 13, 15, 16) |

| 14 | Title + two-column text | 经典的双栏对比或分类内容页 |
| 15 | Title + Picture + Text | 图文混排页，图片位于特定位置 |
| 16 | Title Left + Text Right | 侧边栏式布局，标题位于左侧 |
| 17-23 | Picture Frame | 图片展示页，包含不同样式和标题组合的图片框 |
| 24-25 | Blank | 空白页 |
| 26-31 | Title / Section Slide Gray | 章节过渡页 (变体 1, 2, 3, 4, 5, 6) |
